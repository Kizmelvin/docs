title: "Mobile network providers"
description: "Supported mobile money and telecom operators where Mono has coverage."
---

Mono coverage includes **mobile network operators (MNOs)** and **mobile money** providers in addition to banks. The exact list depends on product and country.

## Where to find the list

- **Coverage page** – The list of supported institutions is on [Coverage](/resources/coverage).
- **Dashboard** – The institution picker in the Mono widget shows currently supported banks and, where applicable, mobile money providers.
- **API** – Use the [bank/institution listing endpoint](/api-reference/miscellaneous) to fetch supported institutions programmatically.

## By product

- **Financial Data (Connect)** – Supported banks and, in some markets, mobile money accounts.
- **Payments (DirectPay / Direct Debit)** – Banks and supported MNO/mobile money channels for debit.
- **Identity (Lookup / Prove)** – Coverage is by country and ID type, not institution-specific.

For the latest list, refer to [Coverage](/resources/coverage) or the API reference.
