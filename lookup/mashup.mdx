---
title: "Mashup Lookup"
description: "Verify NIN, BVN, and date of birth in one API call."
---

Mashup lookup verifies NIN, BVN, and date of birth in a single API call for streamlined KYC. Request requires **bvn**, **nin**, and **date_of_birth** (all mandatory).

For request/response parameters, see [Lookup APIs](/lookup/lookup-apis) and [API reference](/api-reference/lookup).
