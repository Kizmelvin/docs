Mono **Lookup** and **Prove** let you verify users and businesses using BVN, NIN, CAC, TIN, address, documents, and more. Use them for KYC, onboarding, and fraud prevention.

## Lookup vs Prove

|                 | **Lookup APIs**                                                          | **Prove**                                                                                                        |
| --------------- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------- |
| **What**        | Direct API calls to verify a single identifier (e.g. BVN, NIN, CAC).     | Full identity journey: document capture, face match, address verification, often with a hosted or embedded flow. |
| **When**        | You already have an ID (e.g. BVN) and want to verify it or pull details. | You want a full KYC flow with documents and biometrics for higher assurance.                                     |
| **Integration** | Server-to-server API calls.                                              | Initiate via API, user completes in Prove widget or SDK.                                                         |

For a full Prove-powered onboarding flow, see [Identity onboarding](/use-cases/identity-onboarding). For individual Lookup APIs (BVN, NIN, CAC, TIN, etc.), see [Lookup APIs](/lookup/lookup-apis).

## Lookup features

- **Account number** – Validate and verify account numbers.
- **BVN** – Verify Bank Verification Numbers and linked identity.
- **NIN** – Verify National Identification Number and retrieve details.
- **CAC** – Verify company registration with the Corporate Affairs Commission.
- **TIN** – Verify Tax Identification Number and tax-related data.
- **House address** – Confirm address and ownership.
- **International passport** – Validate passport details.
- **Driver's license** – Verify driver's license information.
- **Credit history** – Retrieve credit history from bureaus (with consent).
- **Mashup** – Verify NIN, BVN, and date of birth in one API call.

For request/response details and endpoints, see [Lookup APIs](/lookup/lookup-apis) and the [API reference](/api-reference/full-api-index). For governance (BVN/NIN handling, consent), see [Lookup governance](/trust/lookup).
