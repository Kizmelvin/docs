---
title: "NIN Lookup"
description: "Verify National Identification Number and retrieve identity details."
---

NIN Lookup verifies NIN or vNIN and returns identity details. You can request a **PDF** by including the `output=pdf` query parameter; the response includes a `pdf` object with `jobid` and `status`. Use the Poll NIN Job API to check status and retrieve the PDF URL when ready.

For request/response parameters and PDF flow, see [Lookup APIs](/lookup/lookup-apis) and [API reference](/api-reference/lookup).
