---
title: "SDKs"
description: "Add the Mono Connect widgets to your web or mobile application."
---

Mono SDKs let you embed the **Connect** (and related) widgets in your web or mobile app so users can link their bank accounts, complete payments, or verify identity without leaving your product. Use the **public key** from your [Mono app](/quickstart/create-an-app) when initialising the SDK.

## Official SDKs

Maintained by the Mono team for consistent behaviour and support.

| Platform        | Repository | Description |
|-----------------|------------|-------------|
| **Vanilla JS**  | [withmono/A-sample-widget-setup](https://github.com/withmono/A-sample-widget-setup) | Sample widget setup for web |
| **React**      | [withmono/connect.js](https://github.com/withmono/connect.js) | Connect widget for React apps |
| **Android**    | [withmono/connect-android](https://github.com/withmono/connect-android) | Native Android SDK |
| **Flutter**    | [withmono/connect-flutter](https://github.com/withmono/connect-flutter) | Cross-platform Flutter |
| **React Native** | [withmono/connect-react-native](https://github.com/withmono/connect-react-native) | React Native integration |
| **iOS**        | [withmono/connect-ios](https://github.com/withmono/connect-ios) | Native iOS SDK |

Install and configure the SDK for your stack using the repo’s README, then follow the [Connect quickstart](/quickstart/connect) or [Connect integration](/essentials/mono/connect-integration) for the backend (exchange token, store account ID, call Financial Data APIs).
