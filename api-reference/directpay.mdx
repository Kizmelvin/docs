DirectPay endpoints support one-time payments via the payment widget. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`**.

---

## Initiate payment

**POST** `api.withmono.com/v1/payments/initiate`

Creates a payment link; the user completes authorisation in the widget.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/payments/initiate' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"amount":50000,"currency":"NGN","customer":{"name":"Customer","email":"customer@example.com"},"redirect_url":"https://yourapp.com/done"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/payments/initiate', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({
      amount: 50000,
      currency: 'NGN',
      customer: { name: 'Customer', email: 'customer@example.com' },
      redirect_url: 'https://yourapp.com/done'
    })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "payment_xxxxxxxx",
  "reference": "ref_xxx",
  "link": "https://pay.withmono.com/..."
}
```

## Verify payment

**GET** `api.withmono.com/v1/payments/verify`

Checks payment status by reference.

**Query params:** `reference` (string, required).

**Request**

<CodeGroup>
  ```bash Curl
  curl -X GET 'https://api.withmono.com/v1/payments/verify?reference=ref_xxx' \
    -H 'mono-sec-key: YOUR_SECRET_KEY'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/payments/verify?reference=ref_xxx', {
    method: 'GET',
    headers: { 'mono-sec-key': process.env.MONO_SEC_KEY }
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "payment_xxxxxxxx",
  "reference": "ref_xxx",
  "status": "success",
  "amount": 50000
}
```

## Fetch all payments

**GET** `api.withmono.com/v1/payments`

Lists payments with optional filters (page, start, end, status, account, customer).

**Request**

<CodeGroup>
  ```bash Curl
  curl -X GET 'https://api.withmono.com/v1/payments?page=1' \
    -H 'mono-sec-key: YOUR_SECRET_KEY'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/payments?page=1', {
    method: 'GET',
    headers: { 'mono-sec-key': process.env.MONO_SEC_KEY }
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "data": [],
  "meta": { "page": 1, "total": 0 }
}
```
