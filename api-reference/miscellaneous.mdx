Miscellaneous endpoints include bank coverage and other utility APIs. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`** where required.

## Bank coverage

**GET** `api.withmono.com/v1/misc/banks`

**Request**

<CodeGroup>
  ```bash Curl
  curl -X GET 'https://api.withmono.com/v1/misc/banks' \
    -H 'mono-sec-key: YOUR_SECRET_KEY'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/misc/banks', {
    method: 'GET',
    headers: { 'mono-sec-key': process.env.MONO_SEC_KEY }
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
[
  { "id": "058", "name": "GTBank", "code": "058" },
  { "id": "044", "name": "Access Bank", "code": "044" }
]
```
