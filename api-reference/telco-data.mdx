Telco Data endpoints let you retrieve data from active Nigerian phone numbers. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`** and (after login) the telco token.

## Token exchange

**POST** `api.withmono.com/v1/telco/token`

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/telco/token' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"code":"auth_code_from_login"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/telco/token', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'mono-sec-key': process.env.MONO_SEC_KEY },
    body: JSON.stringify({ code: 'auth_code_from_login' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "token": "telco_token_xxxxxxxx",
  "expires_at": "2025-01-16T10:00:00Z"
}
```

## Other endpoints

**Login** – Start user flow; get auth URL. **Identity** – Subscriber identity. **Balances** – Airtime/data balances. **Transactions** – Transaction history. **Details** – Account details. **OTP verification** – Verify OTP.

For full parameters see [Telco Data](/more-products/telco-data).
