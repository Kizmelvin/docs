---
title: 'Initiate Account Linking'
api: 'POST /v2/accounts/initiate'
description: 'Generate a link for users to connect their bank account.'
---

Initiates the account linking flow. Returns a link to redirect the user to Mono Connect.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v2/accounts/initiate' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"customer":{"name":"Samuel Olamide","email":"samuel@example.com"},"meta":{"ref":"99008877TEST"},"scope":"auth","redirect_url":"https://yourapp.com/callback"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v2/accounts/initiate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'mono-sec-key': process.env.MONO_SEC_KEY },
    body: JSON.stringify({ customer: { name: 'Samuel Olamide', email: 'samuel@example.com' }, meta: { ref: '99008877TEST' }, scope: 'auth', redirect_url: 'https://yourapp.com/callback' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "account_xxxxxxxx",
  "code": "auth_code_xxxxxxxx",
  "link": "https://connect.withmono.com/..."
}
```

For full parameters and errors see [Bank Data API](/api-reference/bank-data).
