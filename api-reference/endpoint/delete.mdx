---
title: 'Unlink Account'
api: 'POST /v1/accounts/{id}/unlink'
description: 'Revoke the link for a connected account.'
---

Unlinks the account so it can no longer be used for data or payments. The account ID is from the linking flow.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/accounts/account_id_here/unlink' \
    -H 'mono-sec-key: YOUR_SECRET_KEY'
  ```
  ```javascript Node
  const response = await fetch(`https://api.withmono.com/v1/accounts/${accountId}/unlink`, {
    method: 'POST',
    headers: { 'mono-sec-key': process.env.MONO_SEC_KEY }
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "status": "success",
  "message": "Account unlinked successfully"
}
```

For full parameters and errors see [Bank Data API](/api-reference/bank-data).
