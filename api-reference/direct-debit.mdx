Direct Debit endpoints support recurring payments: create and manage mandates, then debit linked accounts. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`**.

---

## Create a mandate

**POST** `api.withmono.com/v1/mandates`

Creates a mandate (variable or fixed amount, currency, verification method).

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/mandates' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"customer_id":"customer_xxx","amount_type":"variable","currency":"NGN"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/mandates', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({
      customer_id: 'customer_xxx',
      amount_type: 'variable',
      currency: 'NGN'
    })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "mandate_xxxxxxxx",
  "reference": "ref_xxx",
  "status": "pending_authorisation"
}
```

## Initiate mandate authorisation

**POST** `api.withmono.com/v1/mandates/authorise`

Generates a widget link for the user to approve the mandate.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/mandates/authorise' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"mandate_id":"mandate_xxx","redirect_url":"https://yourapp.com/done"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/mandates/authorise', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({
      mandate_id: 'mandate_xxx',
      redirect_url: 'https://yourapp.com/done'
    })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "link": "https://mandate.withmono.com/...",
  "status": "pending"
}
```

## Retrieve a mandate

**GET** `api.withmono.com/v1/mandates/{id}`

Returns mandate details by ID or reference.

**Path params:** `id` (string, required) – Mandate ID or reference.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X GET 'https://api.withmono.com/v1/mandates/mandate_xxx' \
    -H 'mono-sec-key: YOUR_SECRET_KEY'
  ```
  ```javascript Node
  const mandateId = 'mandate_xxx';
  const response = await fetch(`https://api.withmono.com/v1/mandates/${mandateId}`, {
    method: 'GET',
    headers: { 'mono-sec-key': process.env.MONO_SEC_KEY }
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "mandate_xxxxxxxx",
  "reference": "ref_xxx",
  "status": "active",
  "customer_id": "customer_xxx"
}
```

## Balance inquiry

**POST** `api.withmono.com/v1/mandates/balance`

Checks balance before debiting.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/mandates/balance' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"mandate_id":"mandate_xxx"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/mandates/balance', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({ mandate_id: 'mandate_xxx' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "balance": 50000,
  "currency": "NGN"
}
```

## Debit account

**POST** `api.withmono.com/v1/collections`

Debits the linked account using the mandate.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/collections' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"mandate_id":"mandate_xxx","amount":10000,"narration":"Subscription"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/collections', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({
      mandate_id: 'mandate_xxx',
      amount: 10000,
      narration: 'Subscription'
    })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "collection_xxxxxxxx",
  "status": "processing",
  "reference": "ref_xxx"
}
```
