Lookup endpoints verify identity and business data. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`** (and product-specific headers where required).

---

## Account number resolve

**POST** `api.withmono.com/v1/lookup/account-number/resolve`

Resolves an account number to account name and bank.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/lookup/account-number/resolve' \
    -H 'accept: application/json' \
    -H 'content-type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"account_number":"0123456789","bank_code":"058"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/lookup/account-number/resolve', {
    method: 'POST',
    headers: {
      'accept': 'application/json',
      'content-type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({ account_number: '0123456789', bank_code: '058' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "account_name": "Customer Name",
  "account_number": "0123456789",
  "bank_code": "058"
}
```

## BVN: Initiate

**POST** `api.withmono.com/v2/lookup/bvn/initiate`

Initiates a BVN consent request. Scope defaults to `identity`; use `bank_accounts` to retrieve all accounts connected to the BVN.

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v2/lookup/bvn/initiate' \
    -H 'accept: application/json' \
    -H 'content-type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"bvn":"12345678901","scope":"identity"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v2/lookup/bvn/initiate', {
    method: 'POST',
    headers: {
      'accept': 'application/json',
      'content-type': 'application/json',
      'mono-sec-key': process.env.MONO_SEC_KEY
    },
    body: JSON.stringify({ bvn: '12345678901', scope: 'identity' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "status": "success",
  "id": "bvn_xxxxxxxx"
}
```
