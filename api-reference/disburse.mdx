Disburse endpoints let you send money to multiple recipients from your linked mandate accounts. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`**.

## Create instant disbursement

**POST** `api.withmono.com/v1/disbursements`

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/disbursements' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"source_account_id":"source_xxx","amount":10000,"recipient_account":"1234567890","recipient_bank":"058","narration":"Payment"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/disbursements', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'mono-sec-key': process.env.MONO_SEC_KEY },
    body: JSON.stringify({ source_account_id: 'source_xxx', amount: 10000, recipient_account: '1234567890', recipient_bank: '058', narration: 'Payment' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "disbursement_xxxxxxxx",
  "status": "processing",
  "reference": "ref_xxx"
}
```

## Create source account

**POST** `api.withmono.com/v1/disbursements/source-accounts` – Link a business source account; returns activation URL.

---

## Fetch disbursements

**GET** `api.withmono.com/v1/disbursements` – List disbursements with filters.
