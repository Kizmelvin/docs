CAC (Corporate Affairs Commission) endpoints return Nigerian company registration data. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`**.

## Business search

**POST** `api.withmono.com/v1/cac/business`

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/cac/business' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"rc_number":"123456"}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/cac/business', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'mono-sec-key': process.env.MONO_SEC_KEY },
    body: JSON.stringify({ rc_number: '123456' })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "rc_number": "123456",
  "company_name": "Example Ltd",
  "status": "active"
}
```

## Other endpoints

**Directors** – List directors. **Shareholder** – Shareholder details. **Secretary** – Lookup secretary. **PSC** – Persons with significant control. **Profile** – Combined company profile. **Status report** – CAC status report (PDF).

For full parameters see [CAC / Company search](/more-products/cac-company-search).
