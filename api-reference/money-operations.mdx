Money operations endpoints support subaccounts, payouts, and refunds. Base URL: **`https://api.withmono.com`**. Authenticate with **`mono-sec-key`**.

## Create a subaccount

**POST** `api.withmono.com/v1/subaccounts`

**Request**

<CodeGroup>
  ```bash Curl
  curl -X POST 'https://api.withmono.com/v1/subaccounts' \
    -H 'Content-Type: application/json' \
    -H 'mono-sec-key: YOUR_SECRET_KEY' \
    -d '{"name":"Subaccount Name","split_percentage":10}'
  ```
  ```javascript Node
  const response = await fetch('https://api.withmono.com/v1/subaccounts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'mono-sec-key': process.env.MONO_SEC_KEY },
    body: JSON.stringify({ name: 'Subaccount Name', split_percentage: 10 })
  });
  const data = await response.json();
  ```
</CodeGroup>

**Response (200)**

```json
{
  "id": "subaccount_xxxxxxxx",
  "name": "Subaccount Name",
  "split_percentage": 10
}
```

## Fetch subaccounts

**GET** `api.withmono.com/v1/subaccounts` – List subaccounts.

## Payout and refund

**Payout** – List payouts by status. **Payout transactions** – Get transactions by account and payout ID. **Refund** – Initiate refund for a payment.

For full parameters see [Money operations](/more-products/money-operations) and [Payouts](/payments/payouts).
