---
title: "Mandate Biometric Verification"
description: "Authorise Direct Debit mandates via facial recognition."
---

Mandate biometric verification lets users approve Direct Debit mandates with facial recognition (selfie and liveness) instead of a micro-transfer. In the Create a Mandate API, set verification_method to selfie_verification. Default is transfer_verification.

See [Direct Debit guide](/payments/direct-debit-guide) and [Direct Debit API](/api-reference/direct-debit).
